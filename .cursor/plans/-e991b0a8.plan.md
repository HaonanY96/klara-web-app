# 计划：更新长按菜单与遮罩效果

## 1. 长按菜单扩展 (`app/components/TaskContextMenu.tsx`, `app/components/TaskItem.tsx`)

- 保留原有 Focus / Pin / Delete 操作按钮。
- 新增 “Edit Task” 与 “Add Date” 按钮：
- Edit：触发 `TaskItem` 中新的 `onEditTask` 处理（可打开编辑模式或输入框）。
- Add Date：打开当前任务的日期选择 portal（与现有 Date picker 逻辑复用）。
- 去掉任务文本预览区域，菜单主体只展示操作按钮列表。
- Backdrop 改为半透明黑层（例如 `bg-black/50`），但不使用模糊；点击遮罩关闭菜单。
- 确保菜单出现时不会锁死背景滚动（仅遮罩拦截点击）。

## 2. 日期选择遮罩统一 (`app/components/TaskItem.tsx`)

- Date picker portal 的幕后遮罩同样采用半透明黑层，提供点击空白关闭行为。
- 保持弹窗定位逻辑不变，只调整视觉与遮罩样式，使 Add Date 与 Context Menu 体验一致。
